# 🎮 Game of Life

A Java implementation of Conway's Game of Life.

## 📖 About
Conway's Game of Life is a cellular automaton where each cell lives, dies, or reproduces based on its neighboring cells.

## ✨ Features
- Interactive Game of Life simulation
- Grid-based visualization
- Random initial generation
- Multiple generations

## 🛠️ Built With
- Java
- Object-Oriented Programming (OOP)

## 🚀 How to Run
1. Clone the repository
2. Open the project in your IDE 
3. Run
